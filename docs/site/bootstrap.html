Let's Bootstrap 🥾

A readable, user-friendly Python library for bootstrap statistics and confidence intervals.

To skip straight to the API documentation, click here or select a submodule from the navigation bar on the left.

Overview

Let's Bootstrap is designed to be an educational tool for learning about bootstrapping and different bootstrap confidence intervals. It provides an intuitive API for performing bootstrap analyses with a focus on:

  • Ease of use: Simple, readable syntax for common bootstrap operations
  • Performance: Efficient implementation using NumPy for fast resampling
  • Memory efficiency: Support for lazy evaluation and batch processing
  • Flexibility: Multiple confidence interval methods and customization options
Note

Let's Bootstrap includes SciPy as a dependency, though it doesn't use SciPy's own excellent bootstrap implementations.

SciPy is the workhorse of scientific computing in Python and Let's Bootstrap doesn't intend to compete with it, even for bootstrapping. I was motivated to create this library by a desire to learn more about bootstrapping methods—there are a lot and I actually implemented a few not available in SciPy—and I hope the readable implementations will be just as educational for anyone else curious to learn more about computational statistics.

Features

  • Bootstrap resampling with reproducible results (seed support)
  • Lazy evaluation for memory-efficient processing
  • Batch processing for large datasets
  • Multiple confidence interval methods:
    • Percentile
    • Reverse percentile (basic)
    • Bootstrap-t
    • Bias-corrected (BC)
    • Bias-corrected and accelerated (BCa)
  • Support for both vectorized and loop-based statistic calculation

Installation

pip install lets-bootstrap

Quick Start

import numpy as np
from bootstrap import bootstrap

# Create some example data
data = np.random.normal(loc=5, scale=2, size=100)

# Create bootstrap samples
bs = bootstrap(data, n_resamples=10000, seed=42)

# Calculate the mean of each bootstrap sample
bs_means = bs.apply(np.mean)

# Get 95% confidence interval for the mean
lower, upper = bs_means.ci(alpha=0.05, interval="percentile")
print(f"95% CI for the mean: ({lower:.2f}, {upper:.2f})")

Advanced Usage

Lazy Evaluation

By default, bootstrap samples are only generated when needed:

# Creating the bootstrap object doesn't perform any resampling yet
bs = bootstrap(data, n_resamples=10000)

# Resampling happens here when accessing .samples
first_few_samples = bs.samples[:5]

Memory-Efficient Batch Processing

For large datasets, process bootstrap samples in batches:

# Apply a statistic in batches to avoid memory issues
bs_stats = bs.apply(
    stat_fn=my_complex_statistic, 
    batch_size=1000,
    lazy=False, # compute immediately
)

Combine batches with lazy evaluation to defer batched computation until you need it.

# Apply a statistic in batches to avoid memory issues
bs_stats = bs.apply(
    stat_fn=my_complex_statistic, 
    batch_size=1000,
    lazy=True,
)

# Computation happens, in batches, when accessing the distribution
results = bs_stats.distribution

Different Confidence Interval Methods

# Basic percentile method
ci_percentile = bs_stats.ci(alpha=0.05, interval="percentile")

# Bias-corrected and accelerated method
ci_bca = bs_stats.ci(alpha=0.05, interval="bca")

Fluent Interface

data = np.random.exponential(scale=5, size=100)

lo, hi = bootstrap(data).apply(np.median)bootstrap.ci(method="bca")

 27def bootstrap(
 28    data: np.ndarray | list,
 29    n_resamples: int = 10000,
 30    seed: RNGSeed = None,
 31    lazy: bool = True,
 32) -> BootstrapSamples:
 33    """Create bootstrap samples from the given data.
 34
 35    This function creates bootstrap samples by randomly sampling with replacement
 36    from the provided data.
 37
 38    Args:
 39        data: The dataset to resample from. Can be a numpy array or a list.
 40        n_resamples: Number of bootstrap resamples to generate. Default is 10000.
 41        seed: Random number generator seed for reproducibility. Can be an int,
 42            numpy.random.Generator, numpy.random.RandomState, or None.
 43        lazy: If True, resampling is deferred until the samples are actually
 44            needed. If False, resampling is done immediately. Default is True.

 48        them.
 49

 54        >>> bs = bootstrap(data, n_resamples=1000, seed=42)

 56    """

 58    if not lazy:
 59        bs = bs.materialize()
 60    return bs

 63class BootstrapSamples:
 64    """A container for bootstrap resamples.
 65
 66    This class manages bootstrap resamples created from an original dataset. It supports
 67    lazy evaluation where resamples are only generated when needed and batched processing
 68    for memory efficiency.
 69
 70    Attributes:
 71        data: The original dataset from which resamples are created.
 72        n_resamples: Number of bootstrap resamples to generate.
 73        seed: Random number generator seed for reproducibility.
 74        samples: The resampled samples from data. Generated on first access or with a call
 75            to `materialize`.
 76    """
 77
 78    def __init__(
 79        self, data: np.ndarray, n_resamples: int = 10000, seed: RNGSeed = None
 80    ):
 81        """Initialize a `BootstrapSamples` object.
 82
 83        Args:
 84            data: The original dataset to be resampled.
 85            n_resamples: Number of bootstrap resamples to generate. Default is 10000.
 86            seed: Random number generator seed for reproducibility. Can be an int,
 87                numpy.random.Generator, numpy.random.RandomState, or None.
 88        """
 89        self.data = data
 90        self.n_resamples = n_resamples
 91        self.seed = seed
 92        self._samples: np.ndarray | None = None
 93
 94    @property
 95    def samples(self) -> np.ndarray:
 96        """Generate and return bootstrap resamples.
 97
 98        Returns:
 99            A numpy array of bootstrap resamples with shape (n_resamples, len(data)).
100
101        Note:
102            The samples are generated on first access and cached for subsequent calls.
103        """
104        if self._samples is None:
105            self._samples = bootstrap_resample(
106                data=self.data, n_resamples=self.n_resamples, seed=self.seed
107            )
108        return self._samples
109
110    def batch_samples(self, batch_size: int) -> Generator[np.ndarray]:
111        """Generate bootstrap resamples in batches.
112
113        This method is useful for processing large numbers of resamples without
114        exhausting memory. The final result will be the same as a call to `samples`.
115
116        Args:
117            batch_size: Number of resamples to generate or return in each batch.
118
119        Yields:
120            Batches of bootstrap resamples as numpy arrays.
121
122        Note:
123            If samples have already been generated, this method returns batches
124            of the existing samples. Otherwise, it generates new batches on demand.
125        """
126        rng = _get_rng(self.seed)
127        for i in range(0, self.n_resamples, batch_size):
128            if self._samples is None:
129                if i + batch_size > self.n_resamples:
130                    batch_size = self.n_resamples - i
131                yield bootstrap_resample(
132                    data=self.data, n_resamples=batch_size, seed=rng
133                )
134            else:
135                yield self._samples[i : i + batch_size]
136
137    def materialize(self) -> Self:
138        """Generate all bootstrap resamples immediately.
139
140        Returns:
141            This object, with all samples generated and available in `samples`.
142        """
143        self.samples
144
145        return self

147    def apply(
148        self,
149        stat_fn: Callable,
150        lazy: bool = True,
151        batch_size: int | None = None,
152        how: Literal["vectorized", "loop"] = "vectorized",
153    ):
154        """Apply a statistical function to the bootstrap resamples.
155
156        This method creates a `BootstrapDistribution` by applying the given function to each
157        bootstrap sample.
158
159        Args:
160            stat_fn: The statistical function to apply to each resample.
161            lazy: If True, computation is deferred until the distribution is actually needed.
162                If False, computation is done immediately. Useful with `batch_size` for
163                memory efficiency with large datasets. Default is True.
164            batch_size: Number of resamples to process at once. Useful in combination with
165                `lazy` for memory efficiency with large datasets. Default is None (all at
166                once).
167            how: Method for applying the statistical function. Options are "vectorized",
168                which calls `stat_fn(batch, axis=1)`, and "loop", which calls `np.apply_along_axis(stat_fn, 1, batch)`.
169                Default is "vectorized".
170
171        Returns:
172            A `BootstrapDistribution` object containing the results of applying the statistical
173                function to the resamples.
174        """
175        bd = BootstrapDistribution(
176            stat_fn=stat_fn,
177            sample=self,
178            batch_size=batch_size,
179            how=how,
180        )
181        if not lazy:
182            bd = bd.materialize()
183        return bd

186class BootstrapDistribution:
187    """A container for bootstrap distributions created by applying a statistic to resamples.
188
189    This class manages the computation and storage of statistical values derived from bootstrap
190    resamples. It supports lazy evaluation, batched processing for memory efficiency, and
191    various confidence interval methods.
192
193    Attributes:
194        stat_fn: The statistical function to apply to each bootstrap resample.
195        sample: The `BootstrapSamples` object containing the resamples.
196        theta_hat: The statistic computed on the original dataset.
197        batch_size: Number of resamples to process at once, or `None` to process at once.
198        how: Method for applying the stat_fn to resamples ("vectorized" or "loop").
199        se_fn: Optional function to compute standard error, required for bootstrap t confidence
200            intervals.
201    """
202
203    def __init__(
204        self,
205        stat_fn: Callable,
206        sample: BootstrapSamples,
207        batch_size: int | None = None,
208        how: Literal["vectorized", "loop"] = "vectorized",
209        se_fn: Callable | None = None,
210    ):
211        """Initialize a `BootstrapDistribution` object.
212
213        Args:
214            stat_fn: The statistical function to apply to each bootstrap resample.
215            sample: The `BootstrapSamples` object containing the resamples.
216            batch_size: Number of resamples to process at once. Default is None (all at once).
217            how: Method for applying the stat_fn. Options are "vectorized" (calls
218                stat_fn(batch, axis=1)) or "loop" (calls np.apply_along_axis(stat_fn, 1, batch)).
219                Default is "vectorized".
220            se_fn: Function to compute standard error, required for t-type confidence
221                intervals. Default is None.
222        """
223        self.stat_fn = stat_fn
224        self.sample = sample
225        self.theta_hat = self.stat_fn(self.sample.data)
226        self.batch_size = batch_size
227        self.how = how
228        self.se_fn = se_fn
229        self._distribution: np.ndarray | None = None
230
231    @property
232    def distribution(self) -> np.ndarray:
233        """Generate and return the bootstrap distribution.
234
235        Computes the statistic on all bootstrap resamples, either all at once or in batches
236        depending on batch_size.
237
238        Returns:
239            A numpy array containing the statistic computed on each bootstrap resample.
240
241        Note:
242            The distribution is computed on first access and cached for subsequent calls.
243        """
244        if self._distribution is None:
245            if self.batch_size is None:
246                self._distribution = self._compute(self.sample.samples)
247            else:
248                values_list = []
249                for batch in self.sample.batch_samples(self.batch_size):
250                    values_list.append(self._compute(batch))
251                self._distribution = np.concat(values_list)
252
253        return self._distribution
254
255    def _compute(self, batch: np.ndarray) -> np.ndarray:
256        if self.how == "vectorized":
257            return self.stat_fn(batch, axis=1)
258        elif self.how == "loop":
259            return np.apply_along_axis(self.stat_fn, 1, batch)
260        else:
261            raise ValueError(f"Got unknown value for `how`: {self.how}.")
262
263    def materialize(self) -> Self:
264        """Generate the bootstrap distribution immediately.
265
266        Returns:
267            This object, with the distribution computed and available in `distribution`.
268        """
269        self.distribution
270
271        return self

273    def ci(
274        self,
275        alpha: float = 0.5,
276        interval: Literal[
277            "percentile", "reverse_percentile", "t", "bc", "bca"
278        ] = "percentile",
279        expand: bool = True,
280    ) -> tuple[float, float]:
281        """Compute confidence intervals for the bootstrap distribution.
282
283        Args:
284            alpha: The significance level (0 to 1). Default is 0.5 (50% confidence).
285            how: Method for computing confidence intervals. Options are:
286                - "percentile": Percentile method.
287                - "reverse_percentile": Reverse percentile or "basic" method.
288                - "t": Bootstrap t interval (requires se_fn to be set).
289                - "bc": Bias-corrected interval.
290                - "bca": Bias-corrected and accelerated interval.
291                Default is "percentile".
292            expand: Whether to expand the confidence interval to account for narrowness
293                bias when sample size is small. Default is True.
294
295        Returns:
296            A tuple representing the confidence interval bounds.
297
298        Raises:
299            ValueError: If an unknown method is specified or if se_fn is None when
300                using the bootstrap t interval.
301        """
302        if interval == "percentile":
303            return percentile_ci_from_bootstrap_distribution(
304                self.distribution,
305                alpha,
306                expand,
307                len(self.sample.data) if expand else None,
308            )
309        elif interval == "reverse_percentile":
310            return reverse_percentile_ci_from_bootstrap_distribution(
311                self.distribution,
312                self.theta_hat,
313                alpha,
314                expand,
315                len(self.sample.data) if expand else None,
316            )
317        elif interval == "t":
318            if self.se_fn is None:
319                raise ValueError(
320                    "The se_fn is set to None. Set the se_fn of this object in order to use the bootstrap t interval."
321                )
322            return t_ci_from_bootstrap_distribution(
323                self.distribution, self.theta_hat, self.se_fn, alpha
324            )
325        elif interval == "bc":
326            return bc_ci_from_bootstrap_distribution(
327                self.distribution,
328                self.theta_hat,
329                alpha,
330                expand,
331                len(self.sample.data) if expand else None,
332            )
333        elif interval == "bca":
334            return bca_ci_from_bootstrap_distribution(
335                self.distribution,
336                jackknife_distribution(self.sample.data, self.stat_fn),
337                self.theta_hat,
338                alpha,
339                expand,
340                len(self.sample.data) if expand else None,
341            )
342        else:
343            raise ValueError(f"Got unknown value for `how`: {self.how}.")
